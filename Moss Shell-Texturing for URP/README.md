This is a basic example of the Shell-Texturing, a technique used by many games to simulate grass, fur, moss along other things.
This specific example was built for moss simulation using an URP's Simple-Noise based approach, however the basic concept may be used for other creations.

Esse é um exemplo básico de Shell-Texturing, uma técnica usada por muitos jogos para simular grama, pelos, musgo, entre outras coisas.
Esse exemplo em específico foi criado para simulação de musgos usando uma técnica ligada à Simple-Noise da URP, entretanto o conceito básico pode ser usado para outras criações.
